プロジェクト例: Libft, ft_printf, get_next_line, pushswap

## 🚀 Notable Projects

### 1. **minishell** [![C](https://img.shields.io/badge/C-100%25-success)]()
**Key Features**:
- 🛠 Pipe chaining (`|`) and IO redirections (`>`, `>>`, `<`)
- 📟 Signal handling for Ctrl-C/Ctrl-D/Ctrl-\\
- 🧩 Built-in commands (echo, cd, pwd, export, unset, env, exit)
- Collaborated with other students, leveraging Git for version control and improving teamwork and development skills.

Unix シェルの基本機能を再現した軽量なシェルプログラムです。C言語を用いてゼロから実装し、シェルの仕組みを理解しながら、プロセス管理やパイプ処理などの低レベルなシステムプログラミングを学ぶことを目的としました。また今回、標準で実装されている関数も学習目的で敢えて自作のものを使っております。もちろん終了時、実行時にメモリリークは起きないよう、Valgrindなどのツールを使用してチェックしております。
このプロジェクトは2人で進めたため、モジュール化を徹底し、コマンド実行部分、Parser部分、Tokenizer部分に大きく分け、作業を分担しました。
AST（抽象構文木）を用いた構造的なパーシングを導入し、複雑なコマンド入力に耐えられるようにしています。

### 2. **miniRT** [![C](https://img.shields.io/badge/C-100%25-success)]()
**Key Features**:

📐 Vector and matrix math library from scratch

Intersection calculations for spheres, planes, and cylinders

💡 Phong reflection model for realistic lighting (ambient, diffuse, specular)

🎥 Camera transformations and scene file parsing (.rt)

3DのCG画像を生成するレイトレーシングエンジンです。C言語とグラフィックスライブラリ（minilibx）のみを使用し、ゼロから実装しました。このプロジェクトの目的は、3D空間における物体、光、カメラの相互作用を数学的にモデル化し、ベクトル演算や幾何学の知識を実践的なプログラムに落とし込むスキルを習得することでした。

シーン記述ファイル（.rt）を読み込み、そこに定義された物体や光源情報を基に、カメラからスクリーンの一点（ピクセル）に向けてレイ（光線）を飛ばします。レイと物体の交差判定を行い、最も近い交点の色を照明モデルに基づいて計算することで、最終的な2D画像を生成します。特に、法線ベクトルや反射光の計算など、リアルな質感を表現するための数学的知識が求められました。
